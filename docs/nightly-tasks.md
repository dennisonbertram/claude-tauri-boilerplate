# Nightly Tasks

Tasks for automated agents to pick up during off-hours. Each task should be atomic, well-described, and independently completable.

## Format

```
### Task: [Title]

**Priority**: High | Medium | Low
**Estimated Effort**: Small | Medium | Large
**Prerequisites**: Any requirements before starting
**Description**: What needs to be done
**Acceptance Criteria**: How to know it's done
**Files to Touch**: Expected files involved
```

# Current Intent

## Project
Claude Tauri Boilerplate — desktop chat app for Claude (Tauri + React + Bun).

## Current Feature
**Context Window Visualization** — lets users see context window consumption, what's using it, and how much remains. Inline indicator + detailed modal view.

## Why
Users need visibility into context window usage to know when approaching limits, understand what consumes context, and make informed decisions about conversation length.
